Welcome to a growing learning hub for data analytics. This repository contains foundational study notes, practical Excel guides, and statistics concepts designed to help learners build strong analytical skills.

---

## What this Repo Is For

This repository is a learning archive for:

- learners building a data analytics foundation
- aspiring analysts preparing interview concepts
- non-technical stakeholders who want to understand data analytics topics
- anyone who wants a simple, practical reference for Excel, statistics, and analytics workflows

## Quick Start Guide

1. `01_Introduction` - Learn the basics of analytics, what data roles mean, and how to think about data problems.
2. `02_Excel` - Practice Excel formulas, lookups, and data-cleaning techniques used in real analytics work.
3. `03_Statistics` - Study statistics concepts, probability rules, and hypothesis testing for analytics and interview readiness.
4. Future updates will add SQL, Python, Tableau, Power BI, cloud, and project examples.

## How to Use This Repo

- Read the introduction notes first to understand the analytics workflow.
- Use the `02_Excel` folder for task-based Excel reference and formula examples.
- Review the `03_Statistics` notes for theory, probability, distributions, and practical statistics.
- Check the folder names before opening them - some are currently reserved for future content.
